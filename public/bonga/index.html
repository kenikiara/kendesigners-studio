What exactly is this thing?

Fair question. BONGA is a small Windows program, not a website and not a plugin. Here's the honest breakdown:

01The brain

Speech recognition runs on Whisper, an open-source AI model, executing entirely on your CPU. BONGA ships ready to go and lets you swap models in Settings:

  • Fast (default) · quick answers on modest laptops
  • High quality · more accurate, a bit slower
  • All of them work fully offline

02The cleanup

Raw speech is messy. After transcribing, BONGA:

  • Removes fillers: um, uh, mhh
  • Collapses stutters ("the the" → "the")
  • Capitalizes and punctuates sentences
  • Applies your dictionary corrections
  • Obeys voice commands: "new line", "new paragraph"

03The requirements

  • Windows 10 or 11, 64-bit
  • About 4 GB RAM and 350 MB disk
  • Any microphone, built-in is fine
  • No graphics card, no admin rights
  • Internet only for the first download

Full source code is public on GitHub. Check it, fork it, audit it.

Is it really free?

Right now, yes. Fully. No trial, no locked features, no card. BONGA is young and I want real people using it and telling me what's broken. A paid version may come later; whoever downloads now gets it free.

Windows says "Windows protected your PC". Kwani ni virus?

Not a virus. That's Microsoft SmartScreen, and it flags any new app that hasn't bought a code-signing certificate yet (they cost hundreds of dollars a year). Click More info, then Run anyway. The full source code is public on GitHub if you want to check what it does.

How do I install it?

Download the zip, right-click → Extract All, open the folder, and double-click Bonga.exe. The Home screen will tell you if the speech model needs its one-time download (one click). Then open WhatsApp Web, hold Right Ctrl, and say hi to someone. Tip: in Settings you can tick "Launch BONGA when Windows starts" so it's always ready.

Does it understand Kiswahili?

Yes. It ships with a multilingual model covering Kiswahili and 100+ other languages. Set your language in Settings → Language, or leave it on auto-detect.

Is my voice uploaded anywhere?

No. Transcription happens on your laptop using an open-source model (Whisper). BONGA works with the internet switched off, which is the easiest proof: airplane mode it and try.

The text comes out wrong sometimes. Nifanye nini?

Three fixes, in order: speak a touch slower near the mic; add the names it keeps missing to Dictionary; and if your laptop can afford it, switch to the higher-quality model in Settings → Speech model. Then tell me what it misheard in the feedback box below, real examples are gold.

How do I update or uninstall?

Update: download the new zip from this page and replace the old folder. Uninstall: delete the folder. BONGA keeps its settings in your Windows profile; delete %APPDATA%\Bonga too if you want every trace gone.
